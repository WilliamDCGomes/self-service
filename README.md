# SelfService

## Este aplicativo foi feito na intenção de facilitar e agilizar pedidos de comidas em restaurantes. Ele tem o próposito de substituir o papel de um garçom, sendo possível pedir o que você quer comer direto pelo aplicativo, assim como reservar mesas.
## This application was created with the intention of simplifying and expediting food orders in restaurants. It aims to replace the role of a waiter, allowing you to place food orders directly through the app and even make table reservations.

### Tela de Introdução | Introduction Screen

#### A primeira do aplicativo é uma tela de introdução, onde o usuário possui algumas maneiras de se autenticar no aplicativo.
#### The first screen of the application is an introduction screen, where the user has several ways to authenticate within the app.

![Tela de Introdução](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Tela%20de%20Introdu%C3%A7%C3%A3o.jpg)

### Tela de Login | Login Screen

#### Logo em seguida temos a tela de Login, onde o usuário irá se autenticar no aplicativo.
#### Right after that, we have the Login Screen, where the user will authenticate into the app.

![Tela de Login](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Tela%20de%20Login.jpg)

### Tela de Cadastro | Registration Screen

#### Caso seja o primeiro acesso, o usuário pode solicitar um cadastro, conseguindo assim logar no aplicativo.
#### In case of a first-time user, they can request to create an account, allowing them to log into the application.

![Tela de Cadastro](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Cadastro%20de%20Usu%C3%A1rio.jpg)

### Redefinir Senha | Reset Password

#### Existe também a possibilidade do usuário redefinir a sua senha, caso o mesmo não se lembre da senha cadastrada.
#### There is also the option for the user to reset their password if they don't remember the registered password.

![Redefinir Senha](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Tela%20de%20Redefinir%20Senha.jpg)

### Lista de Produtos | Product List

#### Existem duas abas destinadas a mostrar os produtos, uma delas mostra os produtos em geral, enquanto a outra mostra produtos que estão na promoção naquele dia.
#### There are two tabs dedicated to displaying products. One of them shows the general list of products, while the other displays products that are on sale for that day.

##### Produtos em Geral | General Products

![Produtos em Geral](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Lista%20de%20Produtos.jpg)

##### Produtos em Promoção | Products on Sale

![Produtos em Promoção](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Lista%20de%20Produtos%20em%20Oferta.jpg)

#### Para realizar um pedido, basta clicar em um dos produtos, que irá para tela de detalhes do mesmo.
#### To place an order, you simply need to click on one of the products, which will take you to the details screen for that product.

![Fazendo Pedido](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Visualizando%20Produto.jpg)

### Pedidos | Orders

#### Na terceira aba do aplicativo é mostrado uma lista contendo todos os pedidos que você já realizou, onde você pode acompanhar o status dele.
#### In the third tab of the application, a list is displayed containing all the orders you have placed, allowing you to track their status.

![Pedidos](https://github.com/WilliamDCGomes/self-service/blob/master/ReadmeFiles/Lista%20de%20Pedidos.jpg)

### Reservas de Mesas | Table Reservations

#### Na quarta aba, é onde ficam as mesas que você reservou no restaurante, contendo o número da mesa e quantas cadeiras. Vale dizer que é possível fazer outras reservas na mesma tela e para outros dias também.
#### In the fourth tab, you'll find the tables you have reserved at the restaurant, including the table number and the number of chairs. It's worth mentioning that you can make additional reservations on the same screen and for other days as well.

##### Lista de Reservas | Reservation List

### Perfil | Profile

#### Na última aba, é onde se encontra informações do cadastro do usuário, onde ele pode alterar caso queira. Vale ressaltar que caso o usuário logado seja um administrador, um botão para adicionar novos produtos irá aparecer para ele no aplicativo, possibilitando que ele adicione, edite ou remova seus produtos da maneira que desejar.
#### In the last tab, you'll find user profile information, which the user can modify if desired. It's important to note that if the logged-in user is an administrator, a button to add new products will appear in the application for them. This allows the administrator to add, edit, or remove products according to their preferences.
